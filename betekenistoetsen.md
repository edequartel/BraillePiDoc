# Betekenis toetsen

Here's the information formatted as a Markdown table in Dutch:

| Toetscombinatie | Functie                                        |
|-----------------|-------------------------------------------------|
| RDT             | Rechter duimtoets                               |
| LDT             | Linker duimtoets = wissen ingetypte letter      |
| RMDT            | Rechter middelste duimtoets = geeft altijd de instructie |
| LMDT            | Linker middelste duimtoets = spatiebalk         |
| LDT + LMDT      | Leest de regel cel voor cel, letter voor letter of woord voor woord |
| LDT + RDT       | Wijzigt de snelheid van lezen                   |
| RMDT + RDT      | Wijzigt lezen per woord of per letter           |
| LMDT + RMDT     | Wijzigt de cel-instructie (volwassen, kind, vorm, punten) |
| LDT + RMDT      | Laat de oefeningen vanaf de eerste starten      |
| CURSOR          | Leest wat op de cel staat (afhankelijk van cel-instructie) |
| Bij het typen is LDT | Backspace                                   |
| Bij het typen is LMDT | Spatie                                     |

Laat het me weten als je nog meer aanpassingen of informatie nodig hebt!

Een toetsenbord kan in een USB poort gestoken worden.

Met de functietoets f12 kan er tekst worden ingetypt. Door weer f12 in te typen gaat u terug naar oefeningen.

Met de f11 toets kam het extra geluid aan en uit worden gezet. Als het woord paard wordt uitgesproken dan is ook het geluid van een paard te horen.
